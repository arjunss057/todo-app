### Required python installations

- `pip install fastapi`
- `pip install uvicorn`
- `pip install supabase`
- `pip install dotenv`
- `pip install pydantic`

### To run middleware

- Run in cmd: `uvicorn main:app --reload` inside the main.py directory

### Note

- Change the middleware url in the frontend accordingly
- Use .env file to enter credentials for db
